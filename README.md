## Installation and Execution Instructions

Follow these steps to set up and run the application on your local machine.

### 1. Prerequisites
You must have **Node.js** installed. If you do not have it, download and install it from [nodejs.org](https://nodejs.org/).

### 2. Setup
1. Open your terminal (Mac/Linux) or Command Prompt/PowerShell (Windows).
2. Navigate to the project root directory:
   ```bash
   cd cree-vocabulary-explorer
3. Install the required dependencies with:
    ```bash
   npm install
4. Running the App
Once the installation is finished, start the development server with:
    ```bash
   npm run dev
The terminal will display a local URL (e.g., http://localhost:5173). Open this link in your web browser to view the prototype.
